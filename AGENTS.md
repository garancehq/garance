# Garance — AI Agent Instructions

This file provides the context any AI agent needs to work on this repository.

## Overview

Garance is an open source sovereign BaaS (alternative to Supabase). The MVP has 8 components in a polyglot monorepo (Rust, Go, TypeScript).

## Strict Rules

1. **No AI mentions in commits** — no Co-Authored-By, no "generated by", no "AI-assisted"
2. **Commit format**: gitmoji + conventional commit (`:sparkles: feat(scope): description`)
3. **Do not break existing tests** — verify with `cargo test` / `go test ./...` / `npm test` before committing
4. **Docker required** for integration tests (testcontainers)
5. **Do not modify PostgreSQL schemas** without updating both migrations AND the corresponding store layer

## Inter-Service Communication

```
Client → Gateway (:8080, HTTP)
           ├─ /api/v1/*     → Engine (:5000, gRPC)
           ├─ /auth/v1/*    → Auth (:5001, gRPC)
           └─ /storage/v1/* → Storage (:5002, gRPC)
```

gRPC contracts are defined in `proto/` and generated to:
- Go: `proto/gen/go/`
- Rust: compiled at build time by `tonic-build` (`engine/crates/garance-engine/build.rs`)

## PostgreSQL Schemas

| Schema | Service | Tables |
|--------|---------|--------|
| `garance_auth` | Auth | users, sessions, identities, verification_tokens |
| `garance_storage` | Storage | buckets, files |
| `garance_platform` | (future) | projects, api_keys, quotas |
| `garance_audit` | (future) | events |
| `public` / `project_{id}` | Engine | user-defined tables |

## Error Format (all APIs)

```json
{
  "error": {
    "code": "NOT_FOUND",
    "message": "resource not found",
    "status": 404
  }
}
```

Codes: `VALIDATION_ERROR` (400), `UNAUTHORIZED` (401), `PERMISSION_DENIED` (403), `NOT_FOUND` (404), `CONFLICT` (409), `INTERNAL_ERROR` (500).

## Key Dependencies

### Rust (engine)
- axum 0.8, tonic 0.12, prost 0.13
- tokio-postgres 0.7, deadpool-postgres 0.14
- serde, uuid, chrono

### Go (auth, storage, gateway, cli)
- pgx/v5, google.golang.org/grpc
- golang-jwt/jwt/v5, golang.org/x/crypto (argon2)
- minio-go/v7 (storage), cobra (cli)

### TypeScript (schema, sdk, dashboard)
- Next.js 16, React 19, shadcn/ui, Tailwind CSS v4
- tsup, vitest
